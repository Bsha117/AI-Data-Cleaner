# Legacy-Data-Bridge

A professional utility to bridge the gap between legacy binary data files (.dat) and modern web-ready formats (JSON).

## Features
- Binary-to-JSON conversion.
- Designed for industrial sensor data migration.
- Lightweight and dependency-free.

## Usage
`python dat_to_json.py input.dat output.json`
